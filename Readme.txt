# Doctor Appointment System

This project is a **Doctor Appointment System** developed in PHP. It allows users to book appointments with doctors, manage doctor and patient information

## Features

- **User Registration and Login**
  - Patients can make without registration book appointments.
  - Doctors can log in to manage their schedules and appointments.

- **Appointment Booking**
  - Patients can view available doctors and book appointments.



- **Responsive Design**
  - The system is designed to be responsive and works on desktop and mobile devices.

## Technologies Used

- **Backend:** PHP
- **Frontend:** HTML, CSS, JavaScript (with optional libraries like Bootstrap)
- **Version Control:** Git

## Installation

## How to run the Doctor Appointment Management System Project Using PHP and MySQL

1. Download the zip file

2. Extract the file and copy dams folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/HTML)

4.Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with the name damsmsdb

6. Import damsmsdb.sql file(given inside the zip package in the SQL file folder)

7. Run the script http://localhost/dams
