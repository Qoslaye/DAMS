# Doctor Appointment System

This project is a **Doctor Appointment System** developed in PHP. It allows users to book appointments with doctors and manage doctor and patient information efficiently.

## Features

- **User Registration and Login**
  - Patients can book appointments without registration.
  - Doctors can log in to manage their schedules and appointments.

- **Appointment Booking**
  - Patients can view available doctors and book appointments.

- **Responsive Design**
  - The system is designed to be responsive and works seamlessly on both desktop and mobile devices.

## Usage

### For Patients
1. Book an appointment without registration.
2. View available doctors and book an appointment.
3. View and manage your appointments.

### For Doctors
1. Log in with your credentials.
2. Manage your schedule and appointments.
